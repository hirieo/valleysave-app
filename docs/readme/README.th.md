## ValleySave คืออะไร?

Stardew Valley ไม่มีระบบซิงก์เซฟบนคลาวด์อย่างเป็นทางการระหว่าง PC กับมือถือ **ValleySave มาเติมเต็มช่องว่างนั้น**: แอปจะตรวจหาฟาร์มของคุณ อัปโหลดไปยังโฟลเดอร์ `ValleySave/` ใน **Google Drive ของคุณเอง** แล้วให้คุณดาวน์โหลดไปเล่นต่อบนอุปกรณ์อื่นได้ทุกเครื่อง — Windows, macOS, Linux หรือ Android

- 🔒 **ความเป็นส่วนตัวมาก่อน** — ใช้ scope จำกัด `drive.file` ของ Google: แอปเห็นเฉพาะไฟล์ที่ตัวเองสร้างเท่านั้น ไม่มีวันเข้าถึงส่วนอื่นของ Drive คุณ
- 🖥️ **ข้ามแพลตฟอร์มอย่างแท้จริง** — โค้ดเดียว (Flutter) บิลด์เนทีฟทั้งสี่แพลตฟอร์ม
- 🌍 **14 ภาษา** — ไทย อังกฤษ สเปน บาสก์ ฝรั่งเศส เยอรมัน อิตาลี โปรตุเกส รัสเซีย ยูเครน ญี่ปุ่น เกาหลี และจีน (ตัวย่อและตัวเต็ม)

## ฟีเจอร์

### ซิงก์และย้ายเซฟเกม
- **ตรวจหาฟาร์มอัตโนมัติ** — พบเซฟของคุณบนทุกแพลตฟอร์ม รวมถึง Steam (เนทีฟ, Flatpak และ Snap บน Linux) และโฟลเดอร์ที่ถูกป้องกัน `Android/data` บน Android (โหมด root, [Shizuku](https://shizuku.rikka.app/) หรือสะพานเชื่อมด้วยตนเอง)
- **อัปโหลด / ดาวน์โหลดคลิกเดียว** — แต่ละฟาร์มแสดงว่าฝั่งไหนใหม่กว่า (ในเครื่องหรือ Drive) และแนะนำการกระทำที่ถูกต้อง
- **ป้องกันเซฟที่ไม่สมบูรณ์** — เซฟที่ไฟล์ขาดหายบน Drive จะถูกทำเครื่องหมายและดาวน์โหลดครึ่งๆ กลางๆ ไม่ได้

### ความปลอดภัยของข้อมูล (ส่วนที่เราภูมิใจที่สุด)
- **การแทนที่แบบทรานแซกชัน** — ทุกการดาวน์โหลด นำเข้า และกู้คืน ผ่านขั้นตอน *เตรียม → ตรวจสอบ → สำรอง → สลับ → ยืนยัน* พร้อมย้อนกลับอัตโนมัติ ถ้าเครื่องค้างกลางคัน เซฟต้นฉบับจะไม่ถูกแตะหรือถูกกู้คืนโดยอัตโนมัติ ไม่มีวันมีฟาร์มที่เขียนค้างครึ่งเดียว
- **อัปโหลดแบบอะตอมมิก** — การอัปโหลดเขียนลงโฟลเดอร์ "เจเนอเรชัน" ใหม่ที่ไม่เปลี่ยนแปลง และเผยแพร่ด้วยการอัปเดตไฟล์กำกับครั้งสุดท้ายครั้งเดียว อุปกรณ์อื่นไม่มีวันเห็นเซฟที่อัปโหลดค้างครึ่งเดียว
- **สำรองอัตโนมัติพร้อมนโยบายเก็บรักษา** — สร้างสำเนาที่ตรวจสอบแล้วก่อนแทนที่เซฟทุกครั้ง เก็บสำรองอัตโนมัติล่าสุด 5 ชุดต่อฟาร์ม (สำรองที่ทำเองไม่มีวันถูกลบ)
- **กู้คืนหลังแครช** — โฟลเดอร์ชั่วคราวจากงานที่ถูกขัดจังหวะจะถูกตรวจพบและจัดการอย่างปลอดภัยเมื่อเปิดครั้งถัดไป โดยเลือกสำเนาที่ใช้งานได้เสมอ

### เล่นร่วมกันหลายคน (co-op)
- **ย้ายโฮสต์** — โอนบทบาทโฮสต์ของฟาร์ม co-op ให้ผู้เล่นคนอื่นภายในไฟล์เซฟ พร้อมตรวจสอบความสมบูรณ์ทั้งก่อนและหลัง เล่นฟาร์มเดิมต่อได้แม้โฮสต์ประจำไม่อยู่
- **แชร์ฟาร์มกับเพื่อน** — แชร์ฟาร์มผ่าน Drive ให้ผู้เล่นคนอื่นดาวน์โหลดไปเล่นต่อ พร้อมป้ายบทบาท (อ่านอย่างเดียวหรือซิงก์ได้)
- **แสดงผู้เล่นทุกคน** — การ์ดฟาร์ม co-op แสดงชาวไร่ทุกคน ไม่ใช่แค่โฮสต์

### ความสะดวก
- **เปิดเกมจากแอป** — ตรวจหาและเปิด Stardew Valley บน Windows, macOS และ Linux (Steam เนทีฟ / Flatpak / Snap)
- **นำเข้าเซฟจาก .zip** — ด้วยการป้องกันแบบทรานแซกชันเดียวกับการดาวน์โหลด
- **ตัวอัปเดตในตัว** — ตรวจสอบ GitHub Releases และดาวน์โหลดเวอร์ชันใหม่ให้คุณ
- **อินเทอร์เฟซตามฤดูกาล** — ทั้งอินเทอร์เฟซเปลี่ยนตามฤดูในเกม: กลีบดอกไม้ฤดูใบไม้ผลิ หิ่งห้อยฤดูร้อน ใบไม้ร่วงฤดูใบไม้ร่วง หิมะฤดูหนาว

## วิธีการทำงาน

1. **เชื่อมต่อ** — ผูกบัญชี Google ของคุณเอง (OAuth, เฉพาะ scope `drive.file`)
2. **ตรวจหา** — ValleySave ค้นหาฟาร์มของคุณโดยอัตโนมัติ
3. **ซิงก์** — อัปโหลดฟาร์มไปยังโฟลเดอร์ `ValleySave/` ใน Drive ของคุณ
4. **เล่นต่อได้ทุกที่** — ติดตั้ง ValleySave บนอุปกรณ์อื่น เชื่อมบัญชีเดิม ดาวน์โหลดแล้วเล่น

## คำถามที่พบบ่อย

**Stardew Valley มีคลาวด์เซฟอย่างเป็นทางการระหว่าง PC กับมือถือไหม?**
ไม่มี — เซฟถูกเก็บในเครื่องและไม่มีการซิงก์ข้ามแพลตฟอร์มอย่างเป็นทางการ ValleySave ทำให้ได้โดยใช้ Google Drive ของคุณเอง

**เซฟของฉันปลอดภัยไหม?**
ทุกการกระทำที่ทำลายข้อมูลเป็นแบบทรานแซกชัน: ต้นฉบับถูกสำรองและตรวจสอบก่อนแทนที่ และถูกกู้คืนอัตโนมัติหากมีอะไรผิดพลาด

**เซฟของฉันเก็บอยู่ที่ไหน?**
ในโฟลเดอร์ `ValleySave/` ใน Google Drive *ของคุณเอง* ไม่มีเซิร์ฟเวอร์ ValleySave

**ย้ายฟาร์มระหว่าง Windows กับ Android ได้ไหม?**
ได้ — นั่นคือกรณีการใช้งานหลัก รวมถึง macOS และ Linux ในทุกทิศทาง

## คอมไพล์จากซอร์สโค้ด

ความต้องการ: [Flutter](https://docs.flutter.dev/get-started/install) ≥ 3.12 · Android Studio (สำหรับ Android)

```bash
git clone https://github.com/hirieo/valleysave-app.git
cd valleysave-app

flutter pub get
cp .env.example .env
# เดสก์ท็อป: กรอก GOOGLE_CLIENT_ID และ GOOGLE_CLIENT_SECRET ใน .env
flutter run
```
